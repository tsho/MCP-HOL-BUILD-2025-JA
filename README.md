# Snowflake Cortex AI と Managed MCP サーバーを使用したエージェント駆動ワークフローの構築（日本語版）

## 概要

MCPツールを使用して以下をクエリするヘルスリサーチエージェントを構築します：
- 医学文献のためのPubMed
- 臨床試験情報のための臨床試験データベース

また、MCPツールの説明を改善するために使用できる、AIオブザーバビリティを使用した強力な評価フローも紹介します。

[notebook](./build-and-evaluate-langgraph-agents-with-mcp-tools.ipynb)を開いて、そこに記載されている手順に従って開始してください。

この[video(英語版, 日本語は準備中)](https://www.snowflake.com/en/build/americas/agenda?agendaPath=session/1751697)でハンズオンラボを視聴することもできます。
